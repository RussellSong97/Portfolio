vscode 단축키

1. 파일 찾기
[Ctrl] + [P]
2. 파일내용 검색
[Shift] + [Ctrl] + [F]
3. 프로젝트 실행
[F5]
4. 프로젝트 중지
[Shift] + [F5]
5. 자동 import
[Alt] + [Shift] + [O]
