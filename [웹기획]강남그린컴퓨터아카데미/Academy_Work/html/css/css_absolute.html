Absolute Position

absolute 방식은 요소가 포함된 블록 요소를 기준으로 지정한 오프셋 속성의 값만큼 떨어져서 배치되는 방식을 말한다. 또 문서의 일반적인 흐름을 따라 배치되지 않기 때문에 본문에 있는 다른 요소들의 배치 위치에 영향을 준다.

absolute로 지정한 요소의 상위 요소가 position:static(기본값) 이거나 body 일 경우 absolute로 지정한 요소의 오프셋은 body 태그를 기준으로 배치된다.

상위요소가 body도 position:static도 아니므로 상위요소를 기준으로 배치 된다.
